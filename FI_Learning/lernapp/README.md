# FIAE AP1 Lernapp (Vanilla HTML/CSS/JS)

Ein einziges kleines Projekt (Single Page), kein HTML pro Modul.

## Start

### Schnell (in VS Code)
- Datei [lernapp/index.html](index.html) öffnen.
- Falls Module nicht laden: lokalen Server starten.

### Lokaler Server (empfohlen)
Im Ordner `FIAE_AP1_F26/lernapp`:

- Python: `python -m http.server 5500`
- Dann im Browser öffnen: `http://localhost:5500`

## Funktionen
- Dark/Light Theme (mit Speicherung)
- Modulliste mit Suche
- Modul-Ansicht aus Markdown-Dateien in `../module_001_014/`
- Einfache Hash-Navigation (`#001`, `#002`, ...)

## Nächster Schritt
Lege die weiteren Modul-Dateien `002` bis `014` im Ordner `module_001_014` an, dann erscheinen sie automatisch in der App.
