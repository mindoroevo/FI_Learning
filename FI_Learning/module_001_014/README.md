# FIAE AP1 – Handbuchreihe 001–014

Dieses Verzeichnis ist als **Buchreihe** aufgebaut. Jedes Modul ist ein eigenständiges Kapitel mit:
- Lernzielen und Kompetenzniveau
- Theorieblock mit AP1-Prüfungsbezug
- typischen Fehlerbildern und Gegenstrategien
- Praxisfällen in ansteigendem Schwierigkeitsgrad
- Aufgabenpaket mit Erwartungshorizont/Musterlösung
- Selbstdiagnose und Wiederholungsplan

## Didaktisches Prinzip
1. **Verstehen** (Begriffe, Zusammenhänge, Prüfungslogik)
2. **Anwenden** (Fallarbeit unter Randbedingungen)
3. **Begründen** (technisch, wirtschaftlich, sicherheitsbezogen)
4. **Dokumentieren** (prüfbar, knapp, vollständig)

## Lernempfehlung
- Erst Handbuchmodul vollständig bearbeiten.
- Danach Quizdatei des gleichen Moduls nutzen.
- Erst dann ins nächste Modul wechseln.
